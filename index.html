Penetration Testing Home Lab

Comprehensive Vulnerability Assessment & Ethical Hacking Demonstration

Project Overview

This project demonstrates a comprehensive penetration testing assessment conducted in a controlled lab environment. Using industry-standard tools and methodologies, I successfully identified and exploited 3 critical vulnerabilities in a VulnHub WordPress installation, achieving complete system compromise through a systematic attack chain progression from reconnaissance to privilege escalation.
